4. Zweck der Verarbeitung

Die Verarbeitung der erhobenen Daten dient ausschließlich der Abwicklung von Käufen und der Beantwortung von Anfragen. Eine Auswertung für Marketingzwecke findet nicht statt. Datamining findet nicht statt. Die Daten werden nicht an Dritte weitergegeben oder verkauft.

5. Cookies und Speicherung

Diese Website verwendet keine Tracking- oder Marketing-Cookies. GitHub Pages kann technisch notwendige Cookies zur Sicherheit und zum Betrieb der Website setzen. Es werden unsererseits keine Daten im lokalen Speicher Ihres Endgeräts abgelegt. Die Website ist über HTTPS/TLS verschlüsselt.

Cookies von GitHub Pages: GitHub – Use of Cookies
